# @example

### Syntax

```
@example
  <example_code>
```

### Overview

This tag is used to add a code snippet demonstrating how to use a particular API. This is a "block" tag so it can be written across multiple lines.

### Example(s)

```js
/**
 * @example
 *  goTo('https://www.google.com')
 */
function goTo(link) {
  
}

```
