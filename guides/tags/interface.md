# @interface

### Syntax

`@interface <name>`

### Overview

This tag declares the documented symbol to be an interface.

### Example(s)

You can declare an interface without any code:

```js
/**
 * @interface IRect
 * @property {number} x
 * @property {number} y
 * @property {number} width
 * @property {number} height
 */
```
